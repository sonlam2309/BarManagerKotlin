BarManagerKotlin

Ứng dụng quản lý quán bar được phát triển bằng Kotlin và Firebase Realtime Database.
Công nghệ sử dụng

- Kotlin
- Firebase Realtime Database
- RecyclerView
- Material Design
- Android Studio
 Chức năng

- Quản lý đơn vị
- Quản lý đồ uống
- Nhập hàng
- Xuất hàng
- Lịch sử giao dịch
- Thống kê
- Lợi nhuận
